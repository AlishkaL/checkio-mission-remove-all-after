Not all of the elements are important. What you want to do is to remove all of the elements after given from array.

For the illustration we have an array [1, 2, 3, 4, 5] and we need to remove all elements that go after 3 - which is 4 and 5.

We have two edge cases here: (1) if can't find a cutting element then array shoudn't be changed. (2) if array is empty than it should remain empty

Input: {% if interpreter.slug == "js-node" %}Array{% else %}Iterable{% endif %}.

Output: {% if interpreter.slug == "js-node" %}Array{% else %}Iterable{% endif %}.

Example:

{% if interpreter.slug == "js-node" %} 
removeAllAfter([1, 2, 3, 4, 5], 3) == [1, 2, 3]
removeAllAfter([1, 1, 2, 2, 3, 3], 2) == [1, 1, 2]
{% else %} 
remove_all_after([1, 2, 3, 4, 5], 3) == [1, 2, 3]
remove_all_after([1, 1, 2, 2, 3, 3], 2) == [1, 1, 2]
{% endif %}

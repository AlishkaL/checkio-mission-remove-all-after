那么，我的最终解决方案应该是怎样的呢？

几乎是全部。您只需要将您的代码封装成一个函数表达式，并返回您所填充的列表 -result 。

下面是整个解决方案：

from collections.abc import Iterable


def remove_all_after(items: list[int], border: int) -> Iterable[int]:

    result = []
    for i in items:
        result.append(i)
        if i == border:
            break
            
    return result

您可以尝试 "优化 "上面的代码，将return result 放在if 语句中，而不是break 中。但是，如果border 不在items 中，或者items 完全为空，您的函数将什么也不返回 (None)！

这个解决方案看起来和前面的有点像，但使用的是generator 函数。它在函数 "外部 "产生每个值，当border 是最后产生的值时停止。 

from collections.abc import Iterable


def remove_all_after(items: list[int], border: int) -> Iterable[int]:

    for i in items:
        yield i
        if i == border:
            break

在这个解决方案中，在找到border 的索引后，返回items 的一个片段，包括 或整个items 。 

from collections.abc import Iterable


def remove_all_after(items: list[int], border: int) -> Iterable[int]:

    return items[:items.index(border)+1] if border in items else items

为什么list.index() 方法在这里没有引发异常？因为，尽管写的是命令顺序，if border in items 实际上是先执行的！
